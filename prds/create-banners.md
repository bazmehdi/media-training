# Create an Agent Skill

## Objective
Whenever the keywords `create banners` is used in a prompt, this agent skill should be employed to create 3 banner ads with the following dimensions:
- 970x250 pixels
- 300x600 pixels
- 300x250 pixels
each located within its own folder with the the size appended.

## Example prompt 
`create banners for gingerbread biscuits using the following image`
